# GPT3-Dicord-Bot
Python Discord Bot with OpenAI

# Requirements
```lib openai<br>
python < 3.7.X<br>
Currently I use 3.9 if any errors, delete python versions higher than 3.9.X```

## Deployment

To deploy this project run

```bash
  git clone https://github.com/xDelito/gpt3-dicordbot.git
  cd gpt3-dicordbot
  pip install openai
  python openaibot.py
```
